01. id를 이용한 요소 노드 취득

노드는 DOM 트리를 구성하는 기본 단위이다 HTML문서가 로드되면 브라우저는 이를 파싱하여 DOM트리를 생성하고 그 유형으로는 요소 노드 (p,a 등),속성 노드(class,id 등), 텍스트 노드 등이 있다

Document.getElementById() 메소드는 인수로 전달한 id 속성 값을 갖는 하나의 요소 노드를 탐색하여 반환한다 문서 내의 중복 된 id값을 갖는 html 요소가 여러 개 존재하더라도 에러가 발생하진 않지만 해당 id값을 갖는 첫 번쨔 요소만 반환한다
