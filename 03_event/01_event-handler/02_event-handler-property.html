02. 이벤트 핸들러 프로퍼티 방식

        document, Element 타입의 node 객체는
        이벤트에 대응하는 이벤트 핸들러 프로퍼티를 가지고 있다
        이벤트 핸들러 프로퍼티의 키는 이벤트 핸들러 어트리뷰트와 동일하며
        (on+이벤트 타입),이벤트 핸들러 프로퍼티를 함수에 바인딩하면
        이벤트 핸들러가 등록된다

        이벤트 핸들러 어트리뷰트 방식과 비교했을때 html과
        자바스크립트가 뒤섞이는 문제를 해결할 수 있지만,
        하나의 이벤트 핸들러만 바인딩 할 수 있다는 단점이 있다
