02. 이벤트 핸들러 내부의 this

이벤트 헨들러 어트리뷰트

이벤트 핸들러 어트리뷰트 방식의 경우 이벤트 핸들러에의해
        함수로 호출되고 일반 함수 내부의 this는 전역 객체 window를 가리킨다
        window 객체는 자바스키립트 최 상단의 객체

이벤트 핸들러 프로퍼티, addEventListner

        이벤트 핸들러 프로퍼티 방식과 addEventListner 메소드 방식 모두
        이벤트 핸들러 내부의 this는 이벤트를 바인딩한 dom 요소를 가리킨다
        즉, 이벤트 핸들러 내부의 this 는 이벤트 객체의 currentTarget 프로퍼티와 같다
    
화살표 함수로 정의한 이벤트 핸들러 내부의 this는 상위 스코프의 this를 가리킨다
